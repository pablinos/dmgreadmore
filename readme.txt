A block to link to other posts.

== Description ==

This block allows for the searching and selection of a specific post, which is then displayed with a Read More link

== Installation ==

Activate the plugin and then search for the DMG Read More link in the block selector.

== Frequently Asked Questions ==

= What's with the table emptying when you move to the next page? =

This seems to be a slight limitation with DataViews. As the data is being fetched, the posts data is undefined, but even with it set to the previous posts data it still re-renders.
